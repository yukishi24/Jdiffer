# US-02: JobCenter定義の比較対応

## User Story

- **As a**: 運用担当者  
- **I want**: JobCenterのジョブ・ジョブネット定義も比較できるようにしたい  
- **So that**: JP1以外の環境でも同じ方法で差分確認を行いたい  

---

## Background / Context

- 現場ではJP1だけでなくJobCenterも利用されている  
- ツールごとに比較方法が異なり、運用負荷が高い  
- 同一手順・同一画面で比較できることが求められている  
- 本機能はJP1対応をベースとした拡張対応とする  

---

## Acceptance Criteria (High-level)
※ 実装に依存しない、ユーザー視点の完了条件を書く
- [ ] ユーザーがJobCenter定義を指定して比較できること  
- [ ] JP1比較時と同様の操作手順で利用できること  
- [ ] 追加・変更・削除の差分が識別できること  
- [ ] ジョブおよびジョブネット単位で差分を確認できること  
- [ ] フォーマット不正や読み込み失敗時に適切なエラーが表示されること  

---

## Constraints / Non-goals（Optional）
※ 今回「やらないこと」「対象外」を明示する
- やらないこと：
  - JobCenter定義の編集機能
- 技術的制約：
  - JobCenterの定義フォーマット仕様に依存する
- 将来対応予定：
  - 他ジョブ管理ツールへの拡張
