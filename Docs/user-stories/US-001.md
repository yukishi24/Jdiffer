# US-01: JP1ジョブ・ジョブネット定義の差分比較

## User Story

- **As a**: 運用担当者  
- **I want**: JP1のジョブ・ジョブネット定義を2つ指定して差分比較したい  
- **So that**: 環境間（本番／検証など）の設定差異や変更内容を正確に把握したい  

---

## Background / Context

- リリース前後でJP1定義の差異確認を手作業で行っている  
- 目視やテキスト比較では見落としが発生しやすい  
- 差分確認に時間がかかり、運用負荷が高い  
- **前提条件**
  - 本ツールは**JP1を前提として開発する**
  - 比較対象はJP1のジョブおよびジョブネット定義情報とする  

---

## Acceptance Criteria (High-level)
※ 実装に依存しない、ユーザー視点の完了条件を書く
- [ ] ユーザーが2つのJP1定義を指定して比較できること  
- [ ] 追加・変更・削除の差分が識別できること  
- [ ] ジョブおよびジョブネット単位で差分を確認できること  
- [ ] 設定項目レベルの差分内容を確認できること  
- [ ] 差分が存在しない場合、その旨が明確に表示されること  
- [ ] 不正な入力や読み込み失敗時に、ユーザーに分かる形でエラーが通知されること  

---
